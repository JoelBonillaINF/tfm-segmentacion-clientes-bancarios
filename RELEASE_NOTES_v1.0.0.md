# v1.0.0-tfm

Versión asociada al depósito del Trabajo Fin de Máster.

## Incluye

- notebook end-to-end;
- modelo K-Means final con `k = 4` y pipeline persistido;
- aplicación Streamlit para nuevos clientes;
- ejemplos sintéticos de entrada;
- pruebas funcionales;
- resultados resumidos de validación fuera de muestra;
- documentación visual del flujo metodológico.

## Limitaciones

Prototipo académico ejecutado localmente. No incluye integración con sistemas bancarios, autenticación, API productiva, monitorización de deriva ni validación económica mediante campañas reales.
